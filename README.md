#  The Impact of System Prompt Language on Tourism-Focused Large Language Model Response Quality 
This repository contains all code used during the Global Hyperscale AI Camp 2025 held at Jeju National University. Full poster and presentation can also be found here.

Project Notes:
Data was gathered from Chang Woo Choi's Jeju Tour Tool-Calling Dataset (Pusan National University) on HuggingFace.
Experiments were done on Google Colab with models accessed either via OpenAI API and HuggingFace.
